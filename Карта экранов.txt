Переходы между экранами

Главный экран → Экран входа → Основное меню.
Основное меню → Регистрация пассажиров → Поиск рейса → Поиск пассажира → Ввод данных → Оформление багажа → Выдача посадочного талона.
Основное меню → Управление расписанием рейсов → Список рейсов → Добавление/Редактирование рейса.
Основное меню → Бронирование билетов → Выбор рейса → Ввод данных пассажира → Выбор типа билета → Оплата → Подтверждение бронирования.
Основное меню → Проверка безопасности → Список пассажиров → Статус багажа → Готовность к посадке.
Основное меню → Обработка возвратов → Поиск билета → Условия возврата → Выбор способа возврата → Подтверждение возврата.
Основное меню → Мониторинг задержек → Изменение статуса рейса → Уведомления пассажирам.


Примечания к карте экранов:
Каждый экран включает в себя элементы управления, которые соответствуют действиям пользователя, описанным в сценариях.
Важно предусмотреть в интерфейсе визуальные индикаторы для статусов рейсов, проверок безопасности, статусов багажа и других ключевых данных.
Переходы между экранами должны быть интуитивно понятными, с минимальными кликами для выполнения каждой задачи.
